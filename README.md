# Sales-Dashboard-Project
## A dynamic Excel dashboard showcasing real-time sales performance with interactive visuals. Designed to track KPIs, trends, and insights for smarter business decisions.
## Dataset Used 
- <a href="https://github.com/ShritikU/Sales-Dashboard-Project/blob/main/Sales%20Performance%20and%20Growth%20Analysis%20Dashboard%20Project%202%20Shritik.xlsx">Dataset </a>
## Questions (KPIs)
- What is the total revenue generated this month?
- How much did revenue grow compared to last month?
- How many units were sold this year?
- Which product category contributed the highest sales?
- What is the current conversion rate?
- How many new customers were acquired this month?
- What is the revenue contribution of the top-selling category?
